# Code Directory

## Purpose
Contains all source code, scripts, and notebooks for data analysis and processing.

## Code Organization
```
4_Code/
├── data_collection/   # Scripts for data gathering
├── preprocessing/     # Data cleaning and preparation
├── analysis/          # Statistical and ML analysis
├── visualization/     # Charts, graphs, and dashboards
├── utils/            # Helper functions and utilities
└── notebooks/        # Jupyter notebooks for exploration
```

## Development Guidelines
- Use Python as the primary language
- Follow PEP 8 style guidelines
- Include docstrings for all functions
- Add comments for complex logic
- Version control with Git

## Dependencies
Create a `requirements.txt` file with:
```
pandas
numpy
scikit-learn
matplotlib
seaborn
jupyter
plotly
dash
```

## Getting Started
1. Set up virtual environment
2. Install dependencies: `pip install -r requirements.txt`
3. Start with notebooks for data exploration
4. Move to scripts for production code
