# BigData University Quality Project

## Project Structure

```
BigData_University_Quality/
├── 1_Planning/           # Project planning and documentation
│   ├── Objectifs.docx   # Project objectives and goals
│   ├── Gantt.png        # Project timeline (to be added)
│   └── References.bib   # Bibliography and references
├── 2_Literature/        # Literature review materials
│   ├── Papers/          # Research papers and articles
│   └── Summaries/       # Literature summaries and notes
├── 3_Data/              # Data storage and datasets
├── 4_Code/              # Source code and scripts
└── 5_Report/            # Final reports and documentation
```

## Getting Started

1. **Planning Phase**: Review objectives in `1_Planning/Objectifs.docx`
2. **Literature Review**: Add relevant papers to `2_Literature/Papers/`
3. **Data Collection**: Store datasets in `3_Data/`
4. **Development**: Write code in `4_Code/`
5. **Documentation**: Create final reports in `5_Report/`

## Project Goals

This project aims to leverage big data analytics to assess and improve university quality through:
- Comprehensive data analysis
- Pattern recognition
- Quality metrics development
- Actionable recommendations
